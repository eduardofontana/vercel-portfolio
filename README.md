# Eduardo Fontana | Portfolio

Portfólio pessoal com identidade visual autoral, desenvolvido com Next.js, React, Tailwind CSS e Framer Motion.

## Tecnologias

- **Next.js 16** - Framework React com App Router
- **React 19** - Biblioteca de UI
- **Tailwind CSS 4** - Estilização utilitária
- **Framer Motion** - Animações e motion design
- **TypeScript** - Tipagem estática

## Como executar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) para visualizar o projeto.

## Seções

- **Hero** - Apresentação pessoal
- **Sobre** - Trajetória profissional
- **Skills** - Tecnologias e competências
- **Projetos** - Repositórios do GitHub ordenados por estrelas
- **Contato** - Links e formas de contato

## Deploy

Deploy automático na Vercel via GitHub Actions.
